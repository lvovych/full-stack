Задачи по include, require, cookies, буферизации вывода

Все задания должны выполняться в папке include_require_cookies_ob_start_tasks, в отдельных файлах, которые должны называться по номеру задания, например, "17.php".
Если задание требует создания нескольких файлов - то такие файлы необходимо размещать в папке, название которой равно номеру задания.

Вопросы можно задавать на http://ask.php-academy.kiev.ua/.

1. Создать контактную форму. Сделать так, чтобы один посетитель не мог отправить форму больше 3 раз за одну минуту.

2. Воспроизвести сообщение "headers already sent"

3. Сделать выпадающий список со списком цветов (красный, синий, желтый и т.д.). После выбора цвета и отправки формы надо: 1) запоминать цвет в cookies и выводить на странице текст Lorem Ipsum в выбранном цвете; 2) если был выбран цвет, например, желтый, то при следующей загрузке страницы должен быть тоже выбран желтый в выпадающем списке. То есть, необходимо проставлять selected="seleted" для выбранного пункта в выпадающем списке. То есть, сохранять выбранное значение при перезагрузке страницы.
