Задачи по основам PHP

Все задания должны выполняться в папке php_basics_tasks, в отдельных файлах, которые должны называться по номеру задания, например, "17.php".
Если задание требует создания нескольких файлов - то такие файлы необходимо размещать в папке, название которой равно номеру задания.

Вопросы можно задавать на http://ask.php-academy.kiev.ua/.

1. Создайте переменную name и присвойте ей значение, содержащее ваше имя, например "Иннокентий". Answer

2. Создайте переменную age и присвойте ей значение, содержащее ваш возраст, например 20. Answer

3. Выведите с помощью echo фразу "Меня зовут: %ваше_имя%", например: "Меня зовут: Иннокентий". Answer

4. Выведите фразу "Мне %ваш_возраст% лет", например: "Мне 20 лет". Answer

5. Напишите конструкцию if, которая выводит фразу: "Вам еще работать и работать" при условии, что значение переменной age попадает в диапазон чисел от 18 до 59 (включительно). Answer

6. Расширьте конструкцию if из п.5, выводя фразу: "Вам пора на пенсию" при условии, что значение переменной age больше 59. Answer

7. Расширьте конструкцию if из п.5-6, выводя фразу: "Вам еще рано работать" при условии, что значение переменной age попадает в диапазон чисел от 0 до 17 (включительно). Answer

8. Расширьте конструкцию if из п.5-7, выводя фразу: "Неизвестный возраст" при условии, что значение переменной age является отрицательным числом, или вовсе числом не является. Answer

9. Создайте переменную day и присвойте ей произвольное числовое значение. Answer

10. С помощью конструкции switch выведите фразу: "Это рабочий день", если значение переменной day попадает в диапазон чисел от 1 до 5 (включительно). Answer

11. С помощью конструкции switch выведите фразу: "Это выходной день", если значение переменной day попадает в диапазон чисел от 6 до 7 (включительно). Answer

12. С помощью конструкции switch выведите фразу: "Неизвестный день", если значение переменной day не попадает в диапазон чисел от 1 до 7 (включительно). Answer

13. Заданы две переменные: s - длина участка, который проехал автомобиль; t - время движения в часах. Вывести скорость автомобиля на заданном участке в км/час и в м/сек. Answer

14. Задана переменная $foo = 'bar'; Создать переменную bar, в которой будет храниться число 10. Вывести число 10, используя только переменную $foo (не опечатка). Answer

15. Написать калькулятор. Переменная $a = изменяемое число. Переменная $b = изменяемое число. Переменная $operator = ‘+’ или ‘-’ или ‘/’ или ‘*’ или '%' (остаток от деления). На экран выводить результат в зависимости от значений этих переменных. Не забудьте про деление на 0, если надо - выдавать ошибку что на 0 делить нельзя. Answer

16. Создайте две переменные $a и $b, значения которых будут числами. Выведите максимальное число из этих двух. Answer

17. Создайте две переменные $a = '78' (строковый тип) и $b = 78 (целочисленный тип). Проверьте эти две переменные на равность, с помощью if выведите "равны" или "не равны". Answer

18. Создайте две переменные $a = '78' (строковый тип) и $b = 78 (целочисленный тип). Проверьте эти две переменные на эквивалентность, с помощью if выведите "эквивалентны" или "не эквивалентны". Answer

19. Выведите результат сравнения $a и $b из п.17 с помощью var_dump. Answer

20. Приведите число 20 к типу boolean. Объясните результат. Answer

21. Приведите число 0 к типу boolean. Объясните результат. Answer

22. Приведите число -20 к типу boolean. Объясните результат. Answer

23. Напишите 3 строки, используя три разные функции для вывода текста. Answer

24. Создайте в файле 3 комментария тремя разными типами. Answer

25. Приведите пример, чем отличается <?php от <?=. Answer

26. Объявите константу DAYS_COUNT равную 7 и константу MONTH_COUNT равную 12 двумя разными способами объявления констант. Answer

27. Выполните скрипт по задаче №4 из консоли (необходимо запустить php скрипт по задаче №4 из командной строки). Answer
